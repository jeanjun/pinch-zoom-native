## Installation
<Tabs items={['pnpm', 'npm', 'yarn']} groupId="language">

    ```bash
    pnpm add pinch-zoom-native
    ```  
  </Tab>
  <Tab value="npm">

  </Tab>
  <Tab value="yarn">

  </Tab>

## Usage
```tsx
import { PinchZoom } from 'pinch-zoom-native'

<PinchZoom>
  <img src="image.webp" alt="" />
</PinchZoom>
```
