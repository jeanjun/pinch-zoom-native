## disableDoubleTap
Disables double tap gesture.
```js
instance.disableDoubleTap()
```

## enableDoubleTap
Enables double tap gesture.
```js
instance.enableDoubleTap()
```

## destroy
Cleans up and removes the instance.
```js
instance.destroy()
```
